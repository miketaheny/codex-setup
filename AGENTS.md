# Agent-Flow Adapter for Codex-Compatible Agents

This project uses AF (Agent-Flow) as the canonical workflow.

Read `.agent-flow/config.toml` when present, then follow `AGENT-FLOW.md`. If a repository contains a more specific nested `AGENT-FLOW.md` or `AGENTS.md`, follow the more specific file for that path.

Codex-specific note: this file exists so Codex-compatible agents that auto-load `AGENTS.md` enter the shared Agent-Flow workflow.
