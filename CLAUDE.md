# Agent-Flow Adapter for Claude

This project uses AF (Agent-Flow) as the canonical workflow.

Read `.agent-flow/config.toml` when present, then follow `AGENT-FLOW.md`. If a repository contains a more specific nested `AGENT-FLOW.md` or `CLAUDE.md`, follow the more specific file for that path.

Claude-specific note: this file exists so Claude-compatible agents that auto-load `CLAUDE.md` enter the shared Agent-Flow workflow.
